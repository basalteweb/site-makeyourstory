Du premier message au QR Code

VOUS APPORTEZles souvenirs.

Nous leur donnons une forme. Le but : vous laisser offrir quelque chose de très personnel sans vous perdre dans la technique.

Lancer mon projetPoser une question
01

Vous choisissez l’occasion.

Couple, mariage, naissance, anniversaire, hommage, animal, famille ou idée complètement différente : choisissez ce que vous voulez faire ressentir.

Vous pouvez commencer même si vous n’avez pas encore toutes les photos.
02

Vous décrivez votre idée.

Quelques phrases suffisent : qui reçoit la surprise, pour quelle date, quelle ambiance vous imaginez, et quelles options vous intéressent.

Le formulaire ne vous engage pas à payer immédiatement.
03

Vous envoyez vos éléments.

Après validation, vous recevez les instructions pour transmettre vos photos, vidéos, textes et musiques dans la meilleure qualité possible.

Évitez WhatsApp pour les photos importantes : les fichiers sont souvent compressés.
04

Votre histoire prend forme.

Le site est construit autour de vos contenus : rythme, couleurs, mise en page, message d’ouverture et rendu sur mobile.

Une histoire doit être belle sur téléphone, car c’est là qu’elle sera majoritairement découverte.
05

Vous recevez le lien et le QR Code.

Vous pouvez offrir le QR Code dans une carte ou un coffret. La personne scanne, découvre, et garde le lien.

Pour les projets privés, le lien est communiqué uniquement aux personnes que vous choisissez.
